## 1. 产品概述

邻里狗叫记录分析器是一款帮助小区居民科学记录和分析狗叫时段的工具。用户听到狗叫时一键打点记录，系统自动生成狗叫频次时段分布图，用数据辅助邻里和平沟通。

- 解决问题：仅凭印象说"你家狗总叫"缺乏说服力，需要客观数据支撑
- 目标用户：被狗叫困扰的小区居民、希望文明养犬的宠物主人
- 产品价值：用数据说话，让邻里沟通更理性、更平和

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 记录狗叫、查看分析图表、管理记录数据 |

### 2.2 功能模块
1. **记录主页**：一键打点按钮、今日统计、最近记录列表
2. **分析面板**：24小时时段分布图、星期分布热力图、统计摘要
3. **记录管理**：历史记录列表、编辑/删除记录、添加备注
4. **数据导出**：导出数据为图片或文本，方便沟通使用

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 记录主页 | 一键记录区 | 大尺寸圆形按钮，点击立即记录当前时间戳，带震动反馈 |
| 记录主页 | 今日统计 | 显示今日记录次数、最频繁时段、最近一次记录时间 |
| 记录主页 | 最近记录 | 展示最近10条记录，支持快速删除 |
| 分析面板 | 24小时分布图 | 柱状图展示各小时狗叫频次，高亮峰值时段 |
| 分析面板 | 星期分布图 | 热力图展示一周内各天各时段的分布情况 |
| 分析面板 | 统计摘要 | 总记录数、日均次数、最闹时段、最闹星期几 |
| 记录管理 | 历史列表 | 按日期分组展示所有记录，支持筛选和搜索 |
| 记录管理 | 编辑功能 | 可修改记录时间、添加位置/狗只特征备注 |
| 数据导出 | 分享功能 | 一键生成分析报告图片，包含关键数据和图表 |

## 3. 核心流程

用户听到狗叫 → 点击一键记录按钮 → 系统自动保存时间戳 → 持续记录数天/周 → 进入分析面板查看时段分布 → 识别规律和问题时段 → 导出数据用于邻里沟通

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：温暖琥珀色 (#F59E0B) 搭配柔和米色背景，营造友好平和的氛围
- **辅助色**：薄荷绿 (#10B981) 用于正面数据，珊瑚红 (#EF4444) 标记高频预警时段
- **按钮风格**：大尺寸圆形按钮，带有柔和阴影和按压动效，记录按钮采用渐变填充
- **字体**：标题使用圆润的现代无衬线字体，正文使用清晰易读的字体
- **布局风格**：卡片式布局，圆润边角，柔和阴影，留有充足呼吸空间
- **图标风格**：可爱的卡通小狗图标，表情随记录频次变化（安静→惊讶→困扰）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 记录主页 | 一键记录区 | 居中大圆形按钮，直径 200px+，渐变背景，点击时缩放和波纹动效 |
| 记录主页 | 今日统计 | 三栏数据卡片，数字大号粗体，配简约图标 |
| 记录主页 | 最近记录 | 时间线样式展示，每条记录可左滑删除 |
| 分析面板 | 24小时分布图 | 柱状图，柱子高度代表频次，峰值柱子用特殊颜色高亮 |
| 分析面板 | 星期热力图 | 7x24 网格热力图，颜色深浅代表频次 |
| 分析面板 | 统计摘要 | 徽章式标签展示关键结论，如"晚间 20-22 点最频繁" |
| 记录管理 | 历史列表 | 按日期分组的可折叠列表，支持搜索过滤 |
| 数据导出 | 报告预览 | 生成精美的分享卡片，可保存图片或复制文本 |

### 4.3 响应式
- 采用桌面优先设计，同时完美适配移动端
- 记录按钮在移动端占满屏幕宽度，确保易点击
- 图表在小屏幕上自适应缩放，支持触摸查看详细数据
- 所有交互元素确保足够大的触摸区域（≥44px）

### 4.4 动效设计
- 记录按钮点击：缩放 0.95 → 1.05 → 1.0，配合扩散波纹
- 数据加载：骨架屏渐变动画
- 图表入场：柱子从下往上依次生长，带错落延迟
- 页面切换：淡入淡出 + 轻微位移
- 小狗表情：根据今日记录次数动态变化，增加趣味性
